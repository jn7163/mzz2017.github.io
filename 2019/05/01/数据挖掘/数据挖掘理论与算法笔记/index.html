熵与信息增益

entropy，熵，用来描述数据的不确定性。比方说一堆数据一半是男一半是女，则熵为1；全男全女，熵为0.

H(X)=−∑i=1np(xi)logbp(xi)H(X)=-\sum_{i=1}^np(x_i)log_bp(x_i) H(X)=−​i=1​∑​n​​p(x​i​​)log​b​​p(x​i​​)

我们假定相信这组数据的正确性，则直接判断一个人是男是女概率为0.5，熵是1.0：

H(X)=−0.5∗log20.5−0.5∗log20.5=1.0H(X)=-0.5*log_20.5-0.5*log_20.5=1.0H(X)=−0.5∗log​2​​0.5−0.5∗log​2​​0.5=1.0

假设现在获知其抽不抽烟，那么X分为两个群体

X: \{a=\text{“Non-Smoker”}; \text{b=“Smoker”}\}

H(S∣X=a)=−0.8∗log20.8−0.2∗log20.2=0.7219H(S|X=a) = -0.8*log_20.8-0.2*log_20.2=0.7219H(S∣X=a)=−0.8∗log​2​​0.8−0.2∗log​2​​0.2=0.7219

H(S∣X=b)=−0.05∗log20.05−0.95∗log20.95=0.2864H(S|X=b) = -0.05*log_20.05-0.95*log_20.95=0.2864H(S∣X=b)=−0.05∗log​2​​0.05−0.95∗log​2​​0.95=0.2864

信息增益(Information Gain)：当你知道一个额外的属性时，系统的熵降低的值。也就是这个属性的价值，做决策树的时候要用。

主成分分析（PCA）

特征提取 Feature Extraction

例如三维投射到二维，不同的投射角度造成的信息损失不同。投射即是一种特征提取。

特征提取与特征选择

特征提取和特征选择是DimensionalityReduction（降维）的两种方法，针对于the curse of dimensionality(维灾难)，都可以达到降维的目的。但是这两个有所不同。

特征提取后的新特征是原来特征的一个映射。

特征选择后的特征是原来特征的一个子集。

袁博教授说特征提取包括特征选择，也就是说特征选择是特征提取的一种。综合来看我持保持意见。

根据Variance选择主成分

Variance（方差）：Variance越大代表Information越多，比方说一组身高数据都是170，那么这组身高数据就是没有太多信息的。

很显然如果要从X1和X2中选一个属性的话，就选X1

特征值和特征向量的物理意义

https://baijiahao.baidu.com/s?id=1595375146698848604&wfr=spider&for=pc

顾名思义，特征值和特征向量表达了一个线性变换的特征。在物理意义上，一个高维空间的线性变换可以想象是在对一个向量在各个方向上进行了不同程度的变换，而特征向量之间是线性无关的，它们对应了最主要的变换方向，同时特征值表达了相应的变换程度。

具体的说，求特征向量，就是把矩阵A所代表的空间进行正交分解，使得A的向量集合可以表示为每个向量a在各个特征向量上的投影长度。我们通常求特征值和特征向量即为求出这个矩阵能使哪些向量只发生拉伸，而方向不发生变化，观察其发生拉伸的程度。这样做的意义在于，看清一个矩阵在哪些方面能产生最大的分散度（scatter），减少重叠，意味着更多的信息被保留下来。

P(A,B|G)=P(A|G)P(B|G)\\ P(A|G,B)=P(A|G)\\ P(A,B|G)=P(A,B,G)/P(G)=P(A|B,G)\times P(B,G)/P(G)=P(A|B,G)\times P(B|G)

Sigmoid Threshold Unit

ANN没有使用像阈值开关一样的函数，而是使用了性质很好的sigmoid函数，它的值域是(0,1)，导数在0时最大，且导数为σ(y)⋅(1−σ(y))\sigma(y)\cdot(1-\sigma(y))σ(y)⋅(1−σ(y))，可以很方便地计算出来。下面的算法用到了这个导数性质。

Backpropagation Rule (逆传播算法，BP算法)

对于输出神经元

第jjj个神经元的sigmoid函数的输入是netjnet_jnet​j​​而不是xjix_{ji}x​ji​​。很容易理解，因为第jjj个神经元的工作是输入多个值，输出一个值。

所以说根据sigmoid导数的性质，有∂σ(netj)∂netj=σ(netj)(σ(netj))=oj(1−oj)\frac {\partial \sigma(net_j)}{\partial net_j}=\sigma(net_j)(\sigma(net_j))=o_j(1-o_j)​∂net​j​​​​∂σ(net​j​​)​​=σ(net​j​​)(σ(net​j​​))=o​j​​(1−o​j​​)

看到红框圈起的公式了吗，它和感知机的公式很像，多了一个oj(1−oj)o_j(1-o_j)o​j​​(1−o​j​​)，这恰恰也是sigmoid函数的导数。

说回来，感知机中为什么没有，是因为我们假设输入等于输出，即o(x)=wx
